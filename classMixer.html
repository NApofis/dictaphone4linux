Подробное описание

Класс отвечающий за:

  1. Раздачу буффера устройствам
  2. Преобразование данных с устройств в звук
  3. Накапливание данных
  1. Передачу подготовленных данных (звука) дольше по цепочке для дальнейшего сохранения в файл
